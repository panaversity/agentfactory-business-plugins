# Global Standards Map — Islamic Finance Accounting Frameworks

This reference maps 20 jurisdictions by their primary accounting standard for Islamic financial institutions, the role of AAOIFI, the financial regulator, and unique features relevant to practitioners.

## The Three Accounting Regimes

| Regime                                | Description                                                                                                            | Reference Jurisdictions                                        |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| **AAOIFI Primary**                    | AAOIFI Financial Accounting Standards are the mandatory accounting framework for IFIs                                  | Bahrain, Qatar                                                 |
| **IFRS with Islamic Guidance**        | IFRS is the primary framework; AAOIFI or local bodies provide supplementary Islamic finance guidance                   | Saudi Arabia, UAE, Kuwait, Oman, Pakistan, UK, Nigeria, Turkey |
| **Local Standards (IFRS-equivalent)** | Locally adopted standards substantively equivalent to IFRS, with additional Islamic finance guidance from local bodies | Malaysia (MFRS), Indonesia (PSAK)                              |

## How to Use This Map

1. **Before any Islamic finance exercise**: Look up the jurisdiction to determine which accounting framework applies
2. **For cross-border exercises**: Compare the "Primary Standard" column to identify where framework conflicts arise
3. **For new jurisdiction extensions**: Use this map as the starting point for a jurisdiction overlay SKILL.md file
4. **The key question**: Is this jurisdiction AAOIFI-primary, IFRS-primary, or local-standard? The answer determines which product skill labels to use.
