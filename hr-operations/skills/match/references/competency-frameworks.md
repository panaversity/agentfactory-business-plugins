# Competency Frameworks for Internal Talent Assessment

## Lominger (Korn Ferry Leadership Architect)

- **What it is**: 38 leadership competencies (originally 67, refined) with behavioural indicators at multiple levels
- **Best for**: Large organisations with established competency models; leadership development programmes
- **How readiness maps**: Each competency has "skilled" and "overused" descriptors — readiness = demonstrated "skilled" behaviours in role-critical competencies
- **Limitation**: Requires licensing; can feel prescriptive if applied rigidly

## SHL Universal Competency Framework (UCF)

- **What it is**: 20 competency dimensions grouped into 8 competency factors, mapped to work behaviours
- **Best for**: Structured assessment centres; roles where objectivity and standardisation matter
- **How readiness maps**: Assessment scores map to competency levels (1-5); readiness = meeting threshold on critical competencies
- **Limitation**: Assessment-dependent — requires SHL tools or equivalent for reliable scoring

## DDI (Development Dimensions International)

- **What it is**: Behaviour-based competency system focused on "what good looks like" at each level
- **Best for**: Succession planning specifically; strong on development planning from assessment data
- **How readiness maps**: Uses "ready now / ready in 1-2 years / developing" classification directly
- **Limitation**: Strongest in leadership pipeline; less granular for technical or specialist roles

## When to Use Each

| Context                                | Recommended Framework                                               |
| -------------------------------------- | ------------------------------------------------------------------- |
| Leadership succession                  | DDI — built for this purpose                                        |
| Structured hiring + assessment centres | SHL — standardised and validated                                    |
| Organisation-wide competency model     | Lominger/Korn Ferry — comprehensive                                 |
| No formal framework in place           | Use the six-dimension model in /match as a pragmatic starting point |

## Mapping to /match Readiness Classifications

| Framework Term        | /match Equivalent                                      |
| --------------------- | ------------------------------------------------------ |
| DDI "Ready Now"       | READY NOW                                              |
| DDI "Ready 1-2 Years" | READY IN 6 MONTHS / READY IN 12 MONTHS                 |
| DDI "Developing"      | DEVELOPING                                             |
| SHL "Meets threshold" | READY NOW or READY IN 6 MONTHS (depending on gap size) |
| Lominger "Skilled"    | DEMONSTRATED (in dimension assessment)                 |
| Lominger "Overused"   | Flag — strength becoming a weakness at next level      |

## Development Investment by Readiness

| Readiness          | Investment Type                     | Typical Duration              |
| ------------------ | ----------------------------------- | ----------------------------- |
| READY NOW          | Coaching + stretch assignment       | 0-3 months in-role            |
| READY IN 6 MONTHS  | Targeted skill-building + mentoring | 3-6 months structured plan    |
| READY IN 12 MONTHS | Formal development programme        | 6-12 months with milestones   |
| DEVELOPING         | Long-term pipeline investment       | 18-24 months; re-assess at 12 |
