# Inclusive Language Guide for Job Descriptions

## Gender-Coded Words to Remove

### Masculine-Coded (deter female applicants)

| Avoid                   | Use Instead                           |
| ----------------------- | ------------------------------------- |
| rockstar, ninja, guru   | specialist, expert, lead              |
| dominate, aggressive    | lead, ambitious, driven               |
| crushing it, killing it | delivering results, exceeding targets |
| hunt (for talent/deals) | identify, source, develop             |
| manpower                | workforce, team, staffing             |
| chairman                | chair, chairperson                    |

### Feminine-Coded (can narrow perception of role)

| Avoid as sole descriptor      | Balance with                              |
| ----------------------------- | ----------------------------------------- |
| nurturing                     | supportive and results-oriented           |
| collaborative (as only trait) | collaborative and independently effective |
| warm, gentle                  | professional, approachable                |

Research (Gaucher et al., 2011): Masculine-coded job adverts reduce female applicants by up to 30%, without increasing male applicants.

## Age-Coded Phrases to Remove

| Avoid                             | Use Instead                                     |
| --------------------------------- | ----------------------------------------------- |
| young and dynamic team            | high-performing team                            |
| digital native                    | comfortable with digital tools                  |
| energetic                         | motivated, engaged                              |
| recent graduate (unless required) | entry-level or early-career                     |
| 10+ years experience              | extensive experience / senior-level proficiency |

## Exclusionary Specificity Patterns

| Pattern                                   | Problem                                      | Fix                                                              |
| ----------------------------------------- | -------------------------------------------- | ---------------------------------------------------------------- |
| "5 years in React"                        | Measures tenure, not ability                 | "Strong proficiency in modern frontend frameworks"               |
| "Degree in Computer Science"              | Excludes self-taught and bootcamp graduates  | "Degree in CS or equivalent practical experience"                |
| "Culture fit"                             | Too vague; masks bias                        | Describe specific behaviours: "comfortable with direct feedback" |
| "Startup experience required"             | Excludes corporate talent who may adapt well | "Comfortable in fast-paced, ambiguous environments"              |
| "Must be local" (when remote is possible) | Excludes geographically diverse talent       | State location requirement only if genuinely necessary           |
