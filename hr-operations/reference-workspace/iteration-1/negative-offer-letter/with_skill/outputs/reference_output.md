**This request is outside the scope of the /reference skill.**

You've asked me to draft an offer letter with salary, start date, and terms of employment. The /reference skill handles:

- Reference letters (factual and professional)
- Employment verification letters
- Reference requests

**Offer letters are not reference letters.** An offer letter is a pre-employment document extending a job offer to a candidate. A reference letter is a post-employment (or during-employment) document confirming facts or providing a recommendation about someone who already works or worked here.

### What you should use instead

Use **/draft-offer** — that is the correct skill for drafting offer letters with salary, start date, benefits, and terms of employment.

### Why this matters

Generating an offer letter under the /reference skill would bypass the specific legal and compliance checks that offer letters require (e.g., compensation band validation, benefits eligibility, at-will or fixed-term language, non-compete clauses, contingencies such as background checks). The /draft-offer skill is purpose-built for these requirements.

### Recommended next step

Run `/draft-offer` and provide:
- Candidate name
- Role / job title
- Proposed salary and compensation details
- Proposed start date
- Employment type (full-time, part-time, contract)
- Any specific terms or contingencies (e.g., background check, probation period)
