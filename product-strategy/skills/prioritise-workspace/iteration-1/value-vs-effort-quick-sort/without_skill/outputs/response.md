# Value vs Effort Matrix

Here's a quick 2x2 for your items:

## Matrix

| | Low Effort | High Effort |
|---|---|---|
| **High Value** | Bulk user import, SSO | API webhooks |
| **Low Value** | Dark mode, Notification preferences | Custom dashboard layouts |

## Quadrant Breakdown

### Quick Wins (High Value, Low Effort)
- **Bulk user import** — Essential for enterprise onboarding
- **SSO** — Enterprise requirement

### Major Projects (High Value, High Effort)
- **API webhooks** — Enables integrations but requires event infrastructure

### Nice to Have (Low Value, Low Effort)
- **Dark mode** — Popular request but not a business driver
- **Notification preferences** — Quality of life improvement

### Money Pit (Low Value, High Effort)
- **Custom dashboard layouts** — Lots of work for a niche need

## Recommendation

Start with **Bulk user import** — it's high value and relatively straightforward. Then tackle **SSO** if enterprise deals demand it. Save **API webhooks** for when you have a sprint to dedicate to infrastructure.

Skip **custom dashboard layouts** unless customer research shows stronger demand than expected.
